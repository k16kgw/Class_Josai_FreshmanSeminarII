## Pythonプログラムの正体

これまで Jupyter Notebook では

```python
print("Hello")
```

と書いて **セルを実行**してきた．

しかし実際には，Pythonは

* **テキストファイル（`.py`）**
* **ターミナルからの実行**

が本来の姿である．

---

## `.py` ファイルとは

* Pythonのプログラムを保存した **テキストファイル**
* 拡張子が `.py`

例：`sample.py`

```python
print("Hello, Python")
```

---

## Pythonの実行方法（基本）

### 1. ターミナルでフォルダへ移動

```bash
cd /Users/{ユーザ名}/Documents/Fresh
```

```bash
ls
```

`sample.py` があることを確認する．

---

### 2. Pythonを実行する

```bash
python sample.py
```

または（環境によってはこちら）

```bash
python3 sample.py
```

macOS / Linux では
`python` と `python3` の違いに注意

---

## 実行の流れ（重要）

```text
① エディタで .py を作る
② 保存する
③ ターミナルでその場所へ移動
④ python ファイル名.py
```

「どのフォルダで実行しているか」が最重要

---

## Jupyter Notebook との違い

| 項目    | Jupyter | ターミナル + .py |
| ----- | ------- | ----------- |
| 実行方法  | セルごと    | ファイル単位      |
| 途中実行  | 可能      | 不可          |
| 試行錯誤  | 得意      | 苦手          |
| 自動実行  | 不向き     | 得意          |
| 実務・研究 | 補助      | 主役          |

Jupyter は「ノート」
.py は「プログラム」

## コマンドライン引数（発展）


```text

```

## よくあるエラーと対処法

### ① ファイルが見つからない

```text
python: can't open file 'sample.py'
```

→ **今いるディレクトリが違う**

```bash
pwd
ls
```

---

### ② 文法エラー

```text
SyntaxError
```

→ **Pythonの文法ミス**

* `:` の付け忘れ
* インデントのズレ

---

### ③ NameError

```text
NameError: name 'x' is not defined
```

→ **変数が定義されていない**
