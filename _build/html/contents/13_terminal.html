UNIX系のOSによるPythonの実行方法#

到達目標#

  • ターミナルからPythonプログラムを実行できるようになる．

  • .py ファイルと Jupyter Notebook の違いを理解する．

  • エラーが出たときの見方を知る．

  • 第1回〜第11回までに書いたPythonコードを 「実行可能なプログラム」として扱えるようになる．

第12回の復習#

  • GUI と CUI の違い

  • ターミナルの基本操作

    • pwd, ls, cd, mkdir, rm

  • エディタ

    • vim：最低限の編集

    • VS Code：普段使い

  • **パス（絶対パス・相対パス）**の考え方

第12回は「準備」 第13回は「実行」

Pythonプログラムの正体#

これまで Jupyter Notebook では

print("Hello")

と書いて セルを実行してきた．

しかし実際には，Pythonは

  • テキストファイル（.py）

  • ターミナルからの実行

が本来の姿である．

.py ファイルとは#

  • Pythonのプログラムを保存した テキストファイル

  • 拡張子が .py

例：sample.py

print("Hello, Python")

Pythonの実行方法（基本）#

1. ターミナルでフォルダへ移動#

cd /Users/{ユーザ名}/Documents/Fresh

ls

sample.py があることを確認する．

2. Pythonを実行する#

python sample.py

または（環境によってはこちら）

python3 sample.py

macOS / Linux では python と python3 の違いに注意

実行の流れ（重要）#

① エディタで .py を作る
② 保存する
③ ターミナルでその場所へ移動
④ python ファイル名.py

「どのフォルダで実行しているか」が最重要

演習1（最初の実行）#

  1. ターミナルで /Users/{ユーザ名}/Documents/Fresh に移動せよ．

  2. vim または VS Code で hello.py を作成せよ．

  3. 次の内容を保存せよ．

print("Hello from terminal")

  1. ターミナルで実行せよ．


```bash
python hello.py

コマンドライン引数（発展）#

ターミナルから 値を渡すこともできる．

# args.py
import sys

print(sys.argv)

python args.py 10 20 30

['args.py', '10', '20', '30']

よくあるエラーと対処法#

① ファイルが見つからない#

python: can't open file 'sample.py'

→ 今いるディレクトリが違う

pwd
ls

② 文法エラー#

SyntaxError

→ Pythonの文法ミス

  • : の付け忘れ

  • インデントのズレ

③ NameError#

NameError: name 'x' is not defined

→ 変数が定義されていない

実行結果をファイルに保存（発展）#

python calc.py > result.txt

  • 標準出力をファイルに保存

  • 数値計算・シミュレーションで重要

演習2（これまでのコードを実行）#

第7回〜第11回で作成したプログラムの中から1つ選び， .py ファイルとして保存し，ターミナルから実行せよ．

例：

  • サイコロシミュレーション

  • 数値積分

  • モンテカルロ法

課題（第13回）#

第13回 課題

  1. ターミナルから実行可能なPythonプログラム final.py を作成せよ．

  2. 内容は自由だが，以下を満たすこと：

    • print文を含む

    • for または while を含む

    • 第1回〜第11回で学んだ内容を1つ以上使う

  3. ターミナルで実行した結果が正しく表示されることを確認し， final.py をWebClassから提出せよ．

この講義の意味（重要）#

ターミナルからPythonを実行できるようになった時点で， 「Pythonが書ける」から 「Pythonが使える」段階に到達した

まとめ#

  • Pythonは本来 ターミナルから実行するもの

  • .py ファイルは「実行可能なプログラム」

  • Jupyter と .py は目的によって使い分ける

  • 第13回はこの科目の 到達点
